# rust-cli-test

A simple test RUST cli app

## Install

### Macos

```sh
 $ bash -c "$(curl -fsSL https://raw.githubusercontent.com/rousan/rust-cli-test/master/install.sh)"
```

### Linux

```sh
 $ bash -c "$(curl -fsSL https://raw.githubusercontent.com/rousan/rust-cli-test/master/install.sh)"
```

### Windows

Please download it from [releases](https://github.com/rousan/rust-cli-test/releases) page.
